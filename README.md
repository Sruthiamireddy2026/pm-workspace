## What's here

### `research/`
Competitive analysis across 8 people analytics competitors — Visier, Workday, Dayforce, SAP SuccessFactors, Tableau, Power BI, Looker, and UKG. Built with live web monitoring and structured synthesis, not a one-time snapshot.

### `synthesis/`
Multi-source triangulation across competitive data, customer feedback, and strategy docs. Includes a RICE-scored roadmap cross-reference and feedback synthesis from 45 customer inputs (support tickets, surveys, interviews).

### `transcripts/`
User interview processing. Full session with Rachel Torres, Marketing Ops Manager at GreenPath Studios — themes extracted, quotes surfaced, churn signals identified.

### `week-2/my-prototype/`
End-to-end product artifact build:
- 10-section PRD
- Eval harness: rubric, test cases, thresholds, and product prompt
- LLM-as-judge evaluation run
- Ship verdict: **4.88 / 5.0 overall score**

### `week-3/`
Source data for Beacon Analytics case: competitive landscape CSV, customer feedback themes doc, and FY2026 strategy + OKRs.

### `strategic-brief.md` / `strategic-brief.html`
Product Intelligence Dashboard built by cross-referencing 3 live Google Drive data sources. Outputs: Customer Pain Radar, Competitive Threat Map, OKR Alignment Matrix, and Priority Stack.
